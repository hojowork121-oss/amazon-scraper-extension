# Amazon商品情報抽出ツール（Chrome拡張）

## ■ 概要
Amazonの検索結果ページから、商品情報（商品名・価格・評価・レビュー数・URL）を取得し、CSV形式でダウンロードできるChrome拡張機能です。

ワンクリックでデータを一覧化できるため、EC調査・価格調査・リスト作成業務の効率化を目的として開発しました。

---

## ■ 主な機能
- 検索結果一覧から商品情報を自動取得
- CSVファイルとしてダウンロード
- 広告（スポンサー商品）の除外
- 評価・レビュー数の正確な抽出
- 価格の数値化（￥・カンマ除去済み）

---

## ■ 取得項目
- 商品名
- 価格（数値）
- 評価（例：4.3）
- レビュー数
- 商品URL

---

## ■ 使用方法
1. Chromeに本拡張を読み込む
2. Amazonで検索を実行
3. 拡張アイコンをクリック
4. 「CSVダウンロード」ボタンを押下

---

## ■ 技術構成
- JavaScript（Vanilla JS）
- Chrome Extension（Manifest V3）
- DOM操作（querySelector / querySelectorAll）

---

## ■ 工夫した点
- Amazon特有の複数HTML構造に対応し、安定してデータ取得できるよう調整
- 広告データを除外し、実用的なデータのみ抽出
- 評価・レビューの誤取得を防ぐロジックを実装
- CSVをExcelで扱いやすい形式（BOM付きUTF-8）で出力

---

## ■ 想定利用シーン
- ECサイトの商品リサーチ
- 価格比較・競合分析
- 商品リスト作成業務
- データ収集の自動化
